# Data Source

## Dataset
Help Desk Tickets (Mendeley Data)

## Why this dataset
This project analyzes real-world helpdesk ticket data to evaluate support performance, identify operational bottlenecks, and inform retention risk signals.

## Files used
- issues.csv
- issues_change_history.csv
- issues_snapshots.csv
- (optional) sample_utterances.csv

## License / Attribution
This dataset is used for educational/portfolio purposes with attribution to the original publisher.

## Notes
- Raw data is stored in `data/raw/` and is not committed to Git.
- Cleaned outputs will be stored in `data/processed/`.
